# Maintainers

## Primary maintainer

- [Sarthak Singh (`@iroiro147`)](https://github.com/iroiro147) — project owner and
  primary maintainer

The primary maintainer is responsible for issue triage, pull-request review, source
policy, security response, release decisions, compatibility of the Markdown output
contract, and dependency updates.

Maintainer status is based on sustained project responsibility, not contribution count.
Future maintainers may be added after repeated high-quality contributions and explicit
agreement to take on review, triage, release, and security duties.
